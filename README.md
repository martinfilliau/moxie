moxie
=====

The new Mobile Oxford

Requirements
------------

* Solr 4 BETA
* Redis
* pip (`easy_install pip`)
* bundler (`sudo gem install bundler`)
* node.js and npm

How to run
----------

Installation

* `pip install -r requirements.txt`
* `bundle install`
* npm -g install uglify-js handlebars` (-g means install globally, it will be available from your PATH).

Deployment

* `make static`
